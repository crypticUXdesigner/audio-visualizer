# Audio Files

Place your test audio files (MP3 or OGG) in this directory.

## Usage

1. Place audio files here (e.g., `test1.mp3`, `song.ogg`)
2. Use the file input in the UI to load and play them
3. Or use the console: `loadTestAudio('test1.mp3')`

The visualizer will react to the audio in real-time!
